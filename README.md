# ATS-Friendly Resume Builder

## 📌 Project Overview
ATS-Friendly Resume Builder is a web-based application that helps users create professional resumes optimized for Applicant Tracking Systems (ATS). The platform allows users to enter their personal details, education, skills, and projects through an interactive interface and generate a clean and structured resume suitable for job applications.

## 🚀 Features
- ATS compatible resume format
- Interactive multi-step form
- Real-time resume preview
- Downloadable resume layout
- Responsive design
- Smooth transitions and modern UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
